# Birthday Template

### About

一个简单的生日祝福模板。

生成html，可以直接嵌入邮件。

静态文件位于`https://cdn2.yukisaki.io:41053/birthday_cdn`
，如访问过慢请自行搭建`cdn`。

### 食用方法

```bash
sudo pip3 install flask
python3 ./app.py
```

浏览器打开`127.0.0.1:5000`即可。

修改内容：请自行修改config.json
